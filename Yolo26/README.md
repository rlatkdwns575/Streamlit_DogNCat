# YOLO26 멀티 비전 태스크 앱

Streamlit 기반으로 만든 **YOLO26 멀티태스크 비전 데모 앱**입니다.  
이미지 1장을 업로드하면 선택한 태스크에 따라 객체 탐지, 세그멘테이션, 자세 추정, OBB, 분류 결과를 시각화합니다.

---

## 1. 프로젝트 개요

이 프로젝트는 하나의 웹 앱에서 다양한 컴퓨터 비전 작업을 손쉽게 테스트할 수 있도록 구성한 데모입니다.

사용자는 사이드바에서 원하는 태스크를 선택하고, 신뢰도 임계값(confidence)을 조절한 뒤 이미지를 업로드하면 결과를 바로 확인할 수 있습니다.

지원하는 태스크는 다음과 같습니다.

- **Detection**: 객체 탐지
- **Segmentation**: 객체 분할
- **Pose**: 사람 자세 추정
- **OBB**: 회전 바운딩 박스 탐지
- **Classification**: 이미지 분류

---

## 2. 주요 기능

### Detection
업로드한 이미지에서 객체를 탐지하고,  
각 객체의 클래스명과 신뢰도를 함께 출력합니다.

### Segmentation
객체의 위치뿐 아니라 객체 영역까지 분할하여  
시각적으로 확인할 수 있습니다.

### Pose
사람의 관절 키포인트를 추정하여  
자세 정보를 이미지 위에 표시합니다.  
또한 탐지된 사람 수를 함께 보여줍니다.

### OBB
회전된 객체를 일반 사각형이 아닌  
**Oriented Bounding Box** 형태로 탐지합니다.

### Classification
이미지를 하나의 클래스로 분류하고  
**Top-5 예측 결과**를 확률과 함께 표시합니다.

---

## 3. 실행 화면 구성

- **사이드바**
  - 태스크 선택
  - 신뢰도 조절 슬라이더

- **메인 화면**
  - 이미지 업로드
  - 원본 이미지 표시
  - 추론 결과 이미지 또는 분류 결과 표시
